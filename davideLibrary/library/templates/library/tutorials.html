Q1: I forgot my password, how can I retrieve my account?

To answer this question, please watch the video tutorial and read the text below.

If you’ve forgotten your password, don’t worry! You can easily reset it using the Forgot Password feature located in the bottom corner of the Login page.

  1. Enter Your Username: Start by entering your account username for validation. This step ensures that the system can locate your account.
  2. Request a Temporary Password: Once your username is validated, you can request a temporary password. This temporary password can only be used once, so ensure you copy it using the provided Copy button.
  3. Provide Your UID: After obtaining the temporary password, you will be prompted to enter your UID (Unique ID). Your UID is included with the printed QR code issued to you when your account was created.
    • Scan the QR code using the scanner provided.
    • If the code is valid, the system will confirm it.
  4. Change Your Password: Once the QR code is validated, you will be redirected to the Change Password page. Here, you can securely set a new password for your account.

The steps below will guide you.

Step 1: Click Forgot Password button
Step 2: Enter Account Username
Step 3: Validating Username
Step 4: Copy Temporary Password
Step 5: Validating User QrCode
Step 6: Validating Qrcode
Step 7: Use the Temporary Copied Password
Step 8: Set New Password

Need More Help? If you encounter any issues during this process, please contact your Head Librarian to assist you or issue a new account if necessary.

Q2: How do I set up an Assistant Librarian account?

To answer this question, please watch the video tutorial and read the text below.

To create an Assistant Librarian account, follow these steps:

  1. Navigate to the Registration Page
    • Hover over the Account Registry button, which will redirect you to the Account Registration page.
    • On this page, you can view existing accounts and create new accounts exclusively for Assistant Librarians.
  2. Fill Out the Registration Form
    • Complete the required fields in the registration form.
    • Ensure the Username and Password meet the following requirements:
      • Username Requirements:
        • At least 8 characters long.
        • Must contain at least one number (e.g., 123).
        • Must include at least one uppercase letter (e.g., A).
      • Password Requirements:
        • At least 8 characters long.
        • Must include at least one letter and one number.
        • A password can't be a commonly used password.
        • A password can't be entirely numeric.
        • You can use special characters (e.g., @, $, %).
  3. Pre-Filled Fields
    • The Admin ID and Role fields will be automatically populated.
  4. Confirm the Password
    • Enter the password again in the Confirm Password field to ensure accuracy.
  5. Submit the Form
    • Once the form is successfully submitted, the account will be created and listed in the table below.
  6. Download the Account UID
    • The account's UID (Unique Identifier) will be generated and available for download as a PDF.
    • Important: Ensure the Assistant Librarian securely stores the UID, as it will be needed for account management.

The steps below will guide you.

Step 1: Click Account Registry button
Step 2: Fill out Registration form
Step 3: Take Note of Username requirements
Step 4: Admin ID and Role are automatic fields
Step 5: Enter password and Confirm Registration
Step 6: View account details
Step 7: Download account's pdf copy
Step 8: Print Account with Qrcode Copy

Need More Help? If you encounter any issues during this process, please refer a software maintainance.

Q3: How do I make a student account with printed Qrcode ID?

To answer this question, please watch the video tutorial and read the text below.

To create a Borrower (Student) account, follow these steps:

  1. Navigate to the 'Student Management' Page
    • Hover over the Student Management page.
    • This page will allow you to monitor existing student/borrower accounts.
    • On the left side, you will find the Search and Filter Container, where you can:
      • Search for a student/borrower by scanning their UID (Unique ID).
      • Filter records by grade level.
  2. Understanding the UID
    • A UID is issued when the account is created and is used to identify the borrower/student.
  3. Fill Out the Student/Borrower Registration Form
    • On the right side of the page, you'll find the form for creating a new borrower account.
    • Enter the following required credentials:
      • Borrower's Name (First Name, Middle Initial, Last Name)
      • Grade Level (Choose from 7, 8, 9, 10, 11, or 12)
      • Age
      • Section
      • Adviser
  4. Submit the Form
    • Once you’ve filled out the form, click Save and then Submit.
    • The new account will be listed at the top of the records, indicating a successful submission.
  5. Viewing and Editing Borrower Details
    • After the account is saved, you can view the borrower details in the records table.
    • If you need to make changes, click the 'Edit' button in the Actions field.
  6. Download Borrower Data
    • To download a PDF copy of the borrower’s data:
      • Check the checkbox next to the UID column.
      • Once checked, a 'Download PDF' button will appear.
      • Click the button to start the download.
    • After downloading, you can print the file and give it to the borrower.
  7. Filter and Download Multiple Records
    • You can filter the records by grade level.
    • Once filtered, you can select all the records in the filtered list and download them in a single PDF file.
    • Alternatively, you can choose to download all records without any filtering.

Q4: How do I add or edit a book record?

To answer this question, please watch the video tutorial and read the text below.

To Add a Book to the Inventory:

  1. Navigate to the 'Book Inventory' Page: First, go to the 'Book Inventory' button. This page is responsible for recording books across various categories and types.
  2. Add a New Book: To add a new book, click the 'Add New Book' button located in the actions menu on the right side of the page. This will redirect you to a new page with a form.
  3. Fill Out the Book Details Form: On the form page, you’ll need to fill out the following fields:
    • Category: You can add new categories by clicking the 'Add New Category' button located at the bottom-right corner of the page. The newly added categories will then appear in the category dropdown list.
    • Book Type: The book type field is located in the upper-right corner of the form. Note that you cannot add new book types. You can only select from pre-existing types.
    • Location: The location field also has an 'Add Location' button. You can use this to add new locations if needed.
  4. Submit the Book Information: Once you have completed all required fields, click the 'Create' button to submit the form. You will then be redirected back to the Book Inventory page, where the new record will be listed at the top of the table, indicating successful submission.
  5. Search, Filter, and View Books: Back on the Book Inventory page, you can:
    • Search for a Book: Use the search field to find a book by its book number or title.
    • Filter Books: Filter the book records by category.
    • View Detailed Information: To view detailed information about a book, click the 'Details' button in the actions column.
    • Edit Book Record: To edit a book record, click the 'Edit' button in the actions column. Please note that there are some constraints:
      • If the book has been borrowed, the Category field will be non-editable. All other fields remain editable.
  6. Download or Delete Book Records: - Download PDF: To download a copy of the book record, check the box next to the book(s) you want to download in the 'Select' field on the table. Two buttons will appear:
    • 'Download PDF': Clicking this will download the selected book's data in PDF format.
    • 'Delete': Clicking this will permanently delete the selected book record(s) from the inventory.

The steps below will guide you.

Step 1: Click Book Inventory button
Step 2: Click Add New Book from the Action Menu
Step 3: Fill out the Book Form
Step 4: Click 'Create' to submit the form
Step 5: Successful submission when the record is added on the table
Step 6: Buttons to edit the book, see the book details, download pdf copy, and delete record
Step 7: Downloaded Copy of the book record
Step 8: Delete the record if necessary

Need More Help? If you encounter any issues during this process, please contact your Head Librarian to assist you or issue a new account if necessary.

Q5: How do I add a book borrowing record?

To answer this question, please watch the video tutorial and read the text below.

To create a record for a borrow Slip:
  1. Navigate to the 'Borrowing Records' Page:
    • Go to the 'Borrowing Records' section in the navigation bar. This page keeps track of all the books that have been borrowed by students/borrowers.
    • You will see a table listing all borrowing records, along with an action menu on the right side of the table.
    • Search Options: You can search for a book by:
      • Book Number
      • Borrower UID
      • Librarian who assisted with the borrowing.
    • Filter Options: You can filter the books by borrowed date.
  2. Create a New Borrowing Record:
    • Click the 'Create New Slip' button. This will redirect you to a form for borrowing.
  3. Fill Out the Borrowing Form:
    • The form is located on the right side of the page.
    • Book Number: Scan the book's QR code to enter the book number. The system will automatically calculate the due date based on the book type.
    • Book Type: This determines the borrowing duration.
    • The book details will automatically display in the 'Book Details' panel at the top of the form.
  4. Enter Borrower Details:
    • Borrower Number: Scan the UID of the borrower to enter their borrower number.
    • The borrower details will automatically display in the 'Borrower Details' panel at the lower part of the form.
  5. Check Borrower Status:
    • The borrower's status will automatically be displayed. Borrowers' status can be:
      • Normal: Eligible to borrow.
      • 1st Violation: Eligible to borrow.
      • 2nd Violation: Eligible to borrow, but requires community service as a penalty.
      • 3rd Violation: Not eligible to borrow, but can access books within the library premises. A popup will appear to halt the borrowing process and display a message stating that the borrower is no longer eligible to borrow.
  6. Submit the Borrowing Record:
    • If the borrower is eligible, click the 'Save' button.
    • You will be redirected back to the Borrowing Records page, and the new record will be listed at the top of the table, confirming a successful submission.
  7. Download a Copy of the Record:
    • To download a copy of the record, simply check the box next to the slip number in the table.
    • A 'Download' button will appear in the action menu. Click this button to download a copy of the record.

The steps below will guide you.

Step 1: Click Borrowing Records button
Step 2: The table list
Step 3: The action Menu, Click 'Create New Slip'
Step 4: The borrow slip form
Step 5: Panels for Book Details and Borrower Details
Step 6: Notice for Banned Borrowers
Step 7: Download Pdf button
Step 8: Downloaded Copy of the book slip records

Q6: How do I track all borrowed books?

To answer this question, please watch the video tutorial and read the text below.

To track book records, go to the Book Monitoring in the Navigation bar:
  1. Overview of the Statistics Section
    • Books Borrowed in the Current Month: Displays the total books borrowed during the current month.
    • Ongoing Borrows: Lists books currently with borrowers but not yet returned.
    • Returned Books: Shows the count of books that have been successfully returned.
    • Overdue Books: Highlights books that have exceeded their borrowing duration.
    • Lost Books: Displays the number of books reported as lost.
    • Reserved Books: Shows books that have been reserved by borrowers.
    • Top Books of the Month: Lists the most borrowed books during the current month.
    • 3-Month Line Graph: A visual representation of book borrowing trends over the past three months.
  2. Borrower History Panel
    • Current Borrowed Books: Displays books the borrower is currently borrowing.
    • Returned Books: Lists books successfully returned by the borrower.
    • Overdue Records: Highlights overdue books linked to the borrower.
  3. Record Tables
    • Borrowed Books Table: Lists all books currently on loan.
    • Overdue Books Table: Displays books that have exceeded their borrowing period.
    • Returned Books Table: Shows books that have been successfully returned.
    • Lost Books Table: Highlights books that have been marked as lost.
  4. Filtering Records

    Use the filter fields at the top of the page to quickly locate specific records. Filters include:

    • Book Number: Scan or manually input the book's unique number.
    • Borrower UID: Scan or input the borrower's unique identifier.
  5. Returning Books
    1. Click the 'Return Book' button.
    2. A return form will prompt you to:
      • Enter the Book Number for validation.
      • Mark the book as lost if applicable.
    3. Upon successful return:
      • The record moves to the Returned Books Table.
      • If marked lost, it is added to the Lost Books Table.
  6. Overdue Books
    • Overdue books are listed in the Overdue Table. Actions include:
      • Return Book: Follow the standard return process.
      • Set Penalty: Assign penalties based on the borrower's violation level:
        • Normal: Issue a warning.
        • 1st Violation: Issue another warning.
        • 2nd Violation: Assign community service.
        • 3rd Violation: Ban the borrower from borrowing books but allow in-library access.
  7. Overdue Books from Previous Months

    This section lists books that were not returned in earlier months, ensuring long-term monitoring and follow-up actions.

The steps below will guide you.

Step 1: Book Monitoring Button
Step 2: The Statistics Panel
Step 3: The Borrower History Panel
Step 4: The Borrowed Book List
Step 5: The Overdue Book List
Step 6: The returned and lost book list
Step 7: Overdue book in previous months
Step 8: Penalty form

Q7: How do I make a reservation for books?

To answer this question, please watch the video tutorial and read the text below.

  1. Navigate to the Borrowing Records Page
    • Click the 'Borrowing Records' button in the navigation bar.
    • On this page, you will find a list of records for books that have been borrowed.
  2. Initiate a Reservation
    • In the action menu, click the 'Reserve Book' button.
    • You will be redirected to the Reservation Form page, which is similar in structure to the Borrow Slip Creation Form.
    • The key difference is that submitted reservations will not appear in the Borrowing Records table but will instead be temporarily stored on the 'Book Reservation List' page.
  3. Borrower Status Check
    • If the borrower’s status is marked as banned, they cannot proceed with the reservation process.
  4. Submit the Reservation Form
    • Once the form is completed and submitted:
      • You will be redirected to the 'Book Reservation List' page.
      • This page monitors books that have been reserved.
  5. Reservation Collection Process
    • To collect a reserved book:
      • The borrower must verify their identity by entering or scanning their UID.
    • A reservation is valid for 24 hours from the time it is made:
      • If collected within this time, the record will:
        • Move to the 'Collected Table' in the Reservation List.
        • Be automatically transferred to the Borrowed Book Records table on the Borrowing Records page.
      • If not collected within 24 hours, the reservation will:
        • Be recorded in the 'Expired Reservation Table' on the Reservation List page.

Step 2: Reserve Book and Reservation List buttons
Step 3: Reservation form
Step 4: Detail Panels

Q8: How can i add a new Head Librarian Account?

To answer this question, please read the text below.

  1. Log in as the Super Admin
    • Use the Super Admin account provided by the developers.
    • The Super Admin account is the key account with full access to manipulate core system data.
    • Caution: This account holds significant power. Use it responsibly to avoid unintended system issues.
  2. Navigate to the Database
    • Once logged in, click the gear icon in the top-right corner of the page.
    • This will take you to the Database Page, which is the system's core data storage and management hub.
    • Note: The database page provides access to advanced system capabilities that are not available on regular system pages.
  3. Access the "Custom Users" Section
    • On the database page, go to the "Custom Users" section.
    • Here, you’ll see records of existing user accounts (e.g., assistant librarians).
  4. Add a New User
    • Click the "Add Custom User" button in the top-right corner of the "Custom Users" page.
    • Fill out the user creation form with the following details:
      • Username Requirements:
        • Must be at least 8 characters long.
        • Must include at least one number.
        • Must include at least one uppercase letter.
      • Password Requirements:
        • At least 8 characters long.
        • Must include at least one letter and one number.
        • Cannot be a commonly used password or entirely numeric.
        • Can include special characters (e.g., @, $, %).
      • Admin ID Field:
        • The Admin ID is not automated.
        • Refer to the Admin ID of the first Head Librarian account, then add (+1) to its value.
          Example: If the first account's Admin ID is hpdsnhsAdmin01, the new Admin ID will be hpdsnhsAdmin02.
      • Role Selection:
        • Check "Is Staff" and "Is Active".
        • Do NOT check "Is Superuser"—this will prevent the new account from gaining full Super Admin privileges.
  5. Save the Record
    • Once the form is complete, click Save.
    • The new Head Librarian account will be automatically added to the user list.
  • Important Notes:
    • The Super Admin account is a critical asset. Limit its use to essential administrative actions only.
    • The Database Page is a powerful tool. Handle data with precision and care to maintain system integrity.
    • For assistance, consult the system developers for clarification on sensitive actions.
